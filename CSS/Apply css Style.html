Basic Formatting Using Inline CSS



The tab-size property in CSS is used to set the number of spaces each tab character will display. This method works with pre-formatted text . The tab character can be inserted by holding the Alt key and pressing 0 and 9 together Like other CSS length properties, you can use any unit you want for this. Pixels (px), em, rem, even percentage (%) units will work. Using a relative unit like percentage will change the amount of indentation based on the width of the webpage.
